# AI Office — Chunk 2: Agent Layer

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement all 11 NVIDIA NIM role agents — BaseAgent with retry/timeout/structured output, then each role agent (CTO, Senior Lead, PM, Frontend, Backend, API Engineer, DevOps, QA, Safety, Data Extractor, RAG Search) with role-specific system prompts and output schemas.

**Architecture:** BaseAgent wraps OpenAI-compatible NIM SDK call with 3-retry exponential backoff, 60s timeout, structured JSON output. Each role agent subclasses BaseAgent, injects its system prompt and model ID, parses role-specific output. No agent shares state — all stateless functions.

**Tech Stack:** openai Python SDK (NIM uses OpenAI-compatible API), tenacity (retry), pydantic (output validation), pytest + pytest-asyncio, respx (mock HTTP)

**Prerequisite:** Chunk 1 complete (config, api_key_vault available)

```python
import json
import asyncio
import logging
from openai import AsyncOpenAI
from tenacity import retry, stop_after_attempt, wait_exponential, retry_if_exception_type
from app.config import settings
from app.security.api_key_vault import get_nim_api_key
from app.agents.roles import AgentRole, MODEL_MAP, SYSTEM_PROMPTS

logger = logging.getLogger(__name__)

class BaseAgent:
    def __init__(self, role: AgentRole):
        self.role = role
        self.model = MODEL_MAP[role]
        self.system_prompt = SYSTEM_PROMPTS[role]

    def _client(self) -> AsyncOpenAI:
        return AsyncOpenAI(
            api_key=get_nim_api_key(),
            base_url=settings.NIM_BASE_URL,
            timeout=60.0,
        )

    async def run(self, user_message: str, context: str | None = None) -> dict:
        """Call NIM, retry up to 3x on 5xx, parse JSON output."""
        messages = [{"role": "system", "content": self.system_prompt}]
        if context:
            messages.append({"role": "user", "content": f"Context:\n{context}"})
        messages.append({"role": "user", "content": user_message})

        return await self._call_with_retry(messages)

    @retry(
        stop=stop_after_attempt(3),
        wait=wait_exponential(multiplier=1, min=2, max=10),
        retry=retry_if_exception_type(RuntimeError),
        reraise=True,
    )
    async def _call_with_retry(self, messages: list[dict]) -> dict:
        client = self._client()
        try:
            response = await client.chat.completions.create(
                model=self.model,
                messages=messages,
                temperature=0.2,
                max_tokens=4096,
                response_format={"type": "json_object"},
            )
            content = response.choices[0].message.content
            return json.loads(content)
        except Exception as e:
            logger.warning(f"[{self.role}] NIM call failed: {e}")
            raise RuntimeError(f"NIM API failed for {self.role}: {e}") from e
```
